## Description

Practice jwt authentication using react, react-hook-forms, & bcrypt. The whole purpose of this is just to get a better understanding of the authentication process.

## Usage
```
Install Dependencies:
- npm install

Start Develepment:
- npm run dev

Start UI:
- npm run react:dev

Start Sever:
- npm run server
```
